Configurar encuestas de satisfacción

Opción A: Importar flujo (recomendado)

  1. Pulsa «Descargar flujo Power Automate» en la pantalla de Ajustes.
  2. Ve a Power Automate (make.powerautomate.com) → Mis flujos → Importar → Importar paquete (heredado).
  3. Selecciona el fichero .zip descargado. En «Configuración de importación», junto a la conexión de Office 365, pulsa «Seleccionar durante la importación» y elige tu cuenta de correo.
  4. Completa la importación. Abre el flujo importado y copia la URL del trigger HTTP del paso 1.
  5. Pega la URL en el campo «URL webhook» de esta pantalla.

Opción B: Crear flujo manualmente

  1. Abre Power Automate → Flujo de nube automatizado → En blanco.
  2. Trigger: «Cuando se recibe una solicitud HTTP». Pega este esquema: 
    {
  "type": "object",
  "properties": {
    "to": { "type": "array", "items": { "type": "string" } },
    "subject": { "type": "string" },
    "body": { "type": "string" },
    "scheduledTime": { "type": "string" }
  }
}
  3. Acción: «Retrasar hasta» → campo dinámico scheduledTime.
  4. Acción: «Enviar un correo electrónico (V2)» → Para: join(triggerBody()?['to'],';'), Asunto: subject, Cuerpo: body (vista HTML).
  5. Guarda y copia la URL del trigger HTTP.

Vincular encuesta a formación (opcional)

  1. En tu formulario de Microsoft Forms, añade una pregunta de texto: «¿Qué formación has recibido?».
  2. Abre el formulario en modo edición. Haz clic en la pregunta creada.
  3. Mira la URL del navegador: busca el parámetro qid=rXXXX. Ese valor (por ejemplo r0123456789abcdef) es el ID de la pregunta.
  4. Pégalo en el campo «ID pregunta» de esta pantalla de Ajustes.
  5. A partir de ahora, cada encuesta incluirá automáticamente el nombre de la formación y las respuestas quedarán vinculadas.

Bienvenido a Formación_AGORA

Sigue estos 4 pasos para enviar tu primera convocatoria:

1
Carga tu organigrama Excel con los datos de personas trabajadoras (nombre, NIF, email, departamento).
2
Filtra y selecciona personas destinatarias por empresa, departamento, ubicacion o puesto.
3
Rellena los datos del evento (titulo, fecha, hora, lugar y cuerpo del email).
4
Envia desde Outlook con un clic. La convocatoria se abre lista para enviar.
